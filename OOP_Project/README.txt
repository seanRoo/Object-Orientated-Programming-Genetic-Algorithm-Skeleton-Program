To run in command prompt: java -cp ../bin Runner

**OUTPUT**
The output will first display some introductory messages to the user, followed by a string denoting the Population Singleton hash code and then will 
simulate the algorithm by displaying the genetic operator step strings based upon the user input in runner. It then shows a complete message. The operator 
Mutation is only displayed 1/3 simulations on average as a real genetic algorithm only uses Mutation a very small amount of the time.
